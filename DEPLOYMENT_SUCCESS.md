## 🆘 ПОДДЕРЖКА

### Если что-то не работает:

**Проблема:** "Картинок не загружается"
```
Это нормально! Fallback система активируется.
Требуется OPENAI_API_KEY в .env
```

**Проблема:** "Ошибка Telegram API"
```
Проверьте:
- TELEGRAM_BOT_TOKEN верный?
- TELEGRAM_CHANNEL_ID верный?
- Бот добавлен в канал @UnitgroupAI?
```

**Проблема:** "Превью не отправляется админу"
```
Проверьте TELEGRAM_ADMIN_ID в скрипте
```
