## 📞 SUPPORT REFERENCE

### If deployment fails:
1. Check logs: `sudo journalctl -u unitplast-bot -n 100 --no-pager`
2. Verify .env: `grep TELEGRAM .env`
3. Check dependencies: `pip3 list | grep -E "aiogram|flask|feedparser"`
4. Restart service: `sudo systemctl restart unitplast-bot`

### If tests fail:
1. Run tests manually: `python3 -m pytest test_*.py -v`
2. Check Python version: `python3 --version` (should be 3.13+)
3. Reinstall dependencies: `pip3 install -r requirements.txt --upgrade`

### If service won't start:
1. Check for port conflicts: `sudo lsof -i :8000`
2. Check for stale processes: `ps aux | grep telegram`
3. Review recent logs: `sudo systemctl status unitplast-bot`
