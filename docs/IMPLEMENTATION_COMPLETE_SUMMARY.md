## 📸 КАК РАБОТАЕТ ИНТЕГРАЦИЯ КАРТИНОК

### Primary Method: Парсинг RSS (Экономит токены!)

```python
# Процесс:
1. Загружает RSS-фид источника
2. Ищет записи по ключевым словам поста
3. Извлекает <media_content> или <image> tags
4. Скачивает картинку в data/media_cache/
5. Сохраняет путь в JSON

# Примеры источников:
- IndustryWeek.com/feed
- PlasticsToday.com/feed
- Statista.com (изображения)
- McKinsey.com (графики)
- Gartner.com (диаграммы)
```

### Fallback Method: AI-генерация (Если парсинг не дал)

```python
# Процесс:
1. Если RSS парсинг не нашел картинку
2. Используется AI-промпт из поста
3. Вызывается DALL-E API (требует OpenAI ключ)
4. Генерируется новая картинка за 1-2 минуты
5. Кэшируется локально

# Требует:
- OPENAI_API_KEY в .env
- Интернет соединение
- $0.01-0.05 за генерацию
```

### Пример AI-промптов:

**Post #1 (AI Quality Control):**
```
"Industrial factory conveyor with AI vision system. Red laser inspection 
scanning furniture. AI camera detecting defects. Quality screen showing 
95% accuracy. German facility. Professional lighting."
```

**Post #3 (CNC Machines):**
```
"Collage of 5 CNC machines: German HOMAG, Italian SCM, Chinese WEIKE, 
Korean SHODA. Industrial workshop. Technical specs visible. 
Professional photography."
```
